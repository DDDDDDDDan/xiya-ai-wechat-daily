## [1.0.0] - 2025-11-13

### 新增
- ✨ 首个正式版本发布
- 📱 响应式设计，支持移动端
- 🎨 现代化的卡片式 UI
- 🤖 Python 自动生成脚本
- 📊 自动统计文章数和公众号数
- 🏷️ 智能提取公众号信息
- 🎯 蓝色主题配色方案
- 📝 完整的项目文档

### 特性
- 纯静态页面，无需后端
- 自动按日期排序
- 悬停动画效果
- 公众号名称展示
- 一键生成索引页

### 技术栈
- HTML5 + CSS3
- Python 3.6+
- 无外部依赖
